專案詳情頁

點擊專案後要怎麼呈現完整資訊？

A

獨立路由頁面（React Router）

點擊卡片 → 跳轉到 /project/smartgantt，完整頁面展示。有自己的 URL，可分享連結。按返回鍵回列表。

Pros

  • 可分享專案連結
  • 空間充裕，放多張截圖
  • SEO 友善

Cons

  • 需加 React Router
  • 頁面跳轉感
B

Modal 彈窗覆蓋

點擊卡片 → 彈出全螢幕 modal，背景模糊。不離開頁面，點外面或按 ESC 關閉。

Pros

  • 不需 Router
  • 流暢感好，不離開列表
  • 實作簡單

Cons

  • 無法分享單一專案連結
  • 手機上 modal 體驗較差
C

獨立路由 + URL 可分享（推薦）

用 React Router，但加入滑入動畫（slide transition），兼顧獨立頁面的優點和流暢體驗。URL hash 模式（#/project/smartgantt）不需後端設定。

Pros

  • 可分享連結
  • 動畫過場流暢
  • GitHub Pages 相容（hash router）

Cons

  • 需加 React Router
